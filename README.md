A simple MCP(Model Context protocol) Server built with python that provides real-time weather information for any city using the Open-Metro API.
Features:-
Get current temperature by city name.
Real-time temperature information.
MCP- compliant server implementation.
No API key required.
Lightweight and beginner-friendly project structure.
Easy to set up and run.
